LinkVault 隐私政策

更新日期：2026 年 7 月 18 日

LinkVault 是一款网页 / PWA 书签管理器，并可选配套浏览器扩展。默认本地优先，云端同步与端到端加密（E2E）均为可选功能。本政策说明我们如何处理用户数据。

1. 产品形态

  • Web / PWA — 主应用，可安装到桌面或主屏幕，核心数据在本机浏览器中持久化。
  • 浏览器扩展（可选） — 侧边栏与快捷保存，与主应用共用同一套云端账号（若启用）。

2. 收集的数据

LinkVault 仅处理您主动创建或导入的内容：

  • 书签与分组 — URL、标题、图标、备注、分类、自定义属性、嵌套关系等
  • 可选凭证字段 — 用户名、密码（若填写；启用 E2E 后以密文形式存储）
  • 本地使用统计 — 匿名计数（如保存/导出次数），仅存于本机，不上传

当您选择登录云端同步时，额外处理：

  • 邮箱地址 — 用于 Supabase 邮箱验证码（OTP）登录

当您开启公开分享时，被标记为公开的分组及其书签可被持有链接的访客只读访问（不含需解锁的加密敏感字段）。

3. 数据存储

  • 本地模式（默认） — 书签与设置主要保存在浏览器 localStorage 与 IndexedDB（Dexie）。清除站点数据、卸载 PWA 或清理浏览器存储会删除本地数据。
  • 云端模式（可选） — 登录后，数据同步到您配置的 Supabase 项目。您可在 Dashboard 中查看或删除自己的数据。
  • 端到端加密（可选） — 启用后，敏感字段（如密码）经主密码派生密钥后以 AES-256-GCM 密文存储；主密码不上传。丢失主密码且无 Recovery Key 时，无法恢复已加密字段明文。

4. 数据共享

我们不会将您的数据出售、出租或用于广告定向。数据归属与流向：

  • 本地数据仅存于您的浏览器，默认不传输到服务器
  • 云端数据仅存于您关联的 Supabase 项目，受行级安全策略（RLS）约束
  • 公开分享仅暴露您主动设为公开的内容
  • 错误上报（若配置）可能向 Supabase error_logs 写入匿名技术信息，不包含书签正文

5. 数据控制

您可以随时：

  • 在应用内导出/导入数据，或清空本地存储
  • 通过 Supabase Dashboard 删除云端账号数据
  • 退出登录以停止云端同步（本地数据可保留）
  • 关闭公开分享、关闭 E2E 或重置主密码（按应用内流程操作）

6. 外部服务

  • Supabase — 可选认证、数据库与实时同步。详见 Supabase 隐私政策。
  • 图标服务 — 可能使用公共 favicon/API 按域名拉取图标，通常仅传递域名，不涉及登录身份。

7. 安全性

Web 应用通过 HTTPS 通信；生产环境配置内容安全策略（CSP）等安全头。可选 E2E 在客户端完成加解密。扩展权限按最小必要申请。云端数据受 Supabase RLS 保护。死链检查等边缘能力对私有地址做 SSRF 防护。

8. 政策更新

本隐私政策可能不时更新。重大变更时我们会通过发布说明或仓库更新日志通知。继续使用即表示您知悉更新后的政策。
