LinkVault 隐私政策

更新日期：2026 年 7 月 9 日

LinkVault 是一款书签管理扩展，提供本地存储与可选的云端同步功能。本隐私政策说明我们如何处理用户数据。

1. 收集的数据

LinkVault 在本地模式下仅存储用户主动保存的书签信息：

  • 网页书签 — 您主动保存的网页 URL、标题、图标链接
  • 使用统计 — 本地匿名使用计数（如保存次数、导出次数），仅存于浏览器，不对外发送

当您选择登录云端同步功能时，额外处理：

  • 邮箱地址 — 用于 Supabase 邮箱验证码（OTP）登录

2. 数据存储

  • 本地模式（默认） — 所有书签数据存储在浏览器的 localStorage 中，仅在当前设备可用。清除浏览器数据或卸载扩展将删除这些数据。
  • 云端模式（可选） — 登录后，书签会同步到您自己的 Supabase 数据库。您可以在 Supabase Dashboard 中随时查看或删除自己的数据。

3. 数据共享

我们不会将您的数据出售、出租或分享给任何第三方。您的书签数据归属于您自己：

  • 本地数据仅存于您的浏览器，不传输到任何服务器
  • 云端数据仅存于您关联的 Supabase 项目
  • 我们不收集个人信息用于广告或分析

4. 数据控制

您可以随时：

  • 删除扩展来清除所有本地数据
  • 通过 Supabase Dashboard 删除云端数据
  • 退出登录以停止云端同步，本地数据不受影响

5. 外部服务

本扩展使用了以下外部服务：

  • Supabase — 提供认证与数据库服务。数据存储于 Supabase 的云基础设施中，受 Supabase 的安全措施保护。详见 Supabase 隐私政策。
  • 图标服务 — 非登录状态下使用第三方公共图标 API（xinac.net）获取网站 favicon，仅传递域名，不涉及用户身份。

6. 安全性

扩展权限已最小化：仅申请书签管理、侧边栏、标签页读取等必要权限。所有网络通信均通过 HTTPS 加密。云端数据受 Supabase 行级安全策略（RLS）保护。

7. 政策更新

本隐私政策可能不时更新。重大变更时我们会通过扩展更新日志通知。继续使用即表示您同意更新后的政策。

8. 联系方式

如有任何疑问，可通过 GitHub Issues 联系我们：github.com/HerrHel/HerrHel.github.io。
